# 🤖 AI Chatbot + Lead Generation Automation (n8n + Telegram)

A complete n8n workflow for an AI-powered Telegram chatbot that collects leads, stores them in Google Sheets, and notifies the sales team via email.

---

## 📌 Features

- AI-powered chat using OpenRouter (Gemini)
- Remembers conversations via Google Sheets memory
- Auto-detects lead (name, email, phone)
- Saves leads to Google Sheets
- Sends email notification via Gmail
- Multi-user support
